# LPG Monitoring System

## Overview

This project detects LPG gas leakage using an MQ-2 sensor and alerts the user using a buzzer. It can also send data to an IoT platform for remote monitoring.

## Components

- ESP32
- MQ-2 Gas Sensor
- Buzzer
- LED

## Features

- Gas leakage detection
- Alarm system
- Real-time monitoring
